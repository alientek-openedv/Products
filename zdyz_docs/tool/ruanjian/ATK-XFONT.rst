软件介绍
----------

- 完全免费使用，无任何限制
- 界面简洁、操作简单、功能强大，支持字符模式、图像模式、字库模式
- 支持点阵的字体、大小、移动、旋转、反相、对齐等各种自定义设置
- 支持手动修改点阵数据
- 支持多种方向取模方式，适合各种使用场景
- 支持C51/A51格式输出
- 字库模式也支持点阵的字体、大小、移动、旋转、反相、对齐等各种自定义
- 支持自动升级功能
