软件介绍
----------

- 支持.c/.txt/.h文件里面的数组转换成bin文件
- 支持按小端模式/大端模式转换
- 支持混合数据格式数组转换
- 支持自动识别数组类型
- 支持拖拽方式打开文件
- 支持正点原子软件仓库
