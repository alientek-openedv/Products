软件介绍
----------

- 支持串口接收JPEG数据流并显示，串口波特率可自定义
- 支持网络接收JPEG数据流并显示，支持TCP Server和Client模式
- 支持4种图像显示模式（普通、拉升、居中、放大）
- 支持保存图像截图到指定文件夹
- 支持帧率、帧大小、图片分辨率等信息显示
- 支持简体中文、繁体中文和英文三种语言
- 支持正点原子软件仓库
