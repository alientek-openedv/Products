软件介绍
----------

- 支持UDP/TCP Server/TCP Client三种通信协议
- 支持数据来源显示、时间戳显示
- 支持接收数据转向文件存储
- 支持HEX显示/发送
- 支持窗口保存
- 支持目标主机设置（UDP模式）
- 支持连接列表显示，可指定连接对象通信
- 支持文件发送
- 支持数据流循环发送
- 支持发送/接收计数
- 支持发送区/接收区的字体大小、颜色、背景色设置
- 支持简体中文、繁体中文和英文三种语言
- 支持正点原子软件仓库
